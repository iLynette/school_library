# School Library

> OOP School Library is a simple project that enables you to add Person, Student and Teacher classes.


## Built With

- Ruby


## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- A text editor(preferably Visual Studio Code)

### Install
- [Git](https://git-scm.com/downloads)
- [Ruby](https://www.ruby-lang.org/en/documentation/installation/)

### Usage

#### Clone this repository
```bash
$ https://github.com/iLynette/school_library.git
$ cd school_library
```
